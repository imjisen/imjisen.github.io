创建 GitHub 仓库

你需要一个GitHub仓库用于存放博客的静态页面文件。注意，本文并没有用到GitHub Pages功能，因此仓库名可随意取。（如果不知道GitHub是什么请百度）

创建一个tokens

你需要一个Tokens用于Gridea的同步。

1.访问https://github.com/settings/tokens

2.创建新Tokens

在创建Tokens时，名字可随意，只需勾选repo相关权限即可，如下图。

下载 Gridea

Gridea 官网，请选择对应的系统版本下载并安装。

设置 Gridea

按照下图填写好相应信息（前面几步获取的仓库名和Tokens）

同步到 GitHub

设置好上一步的信息并检测远程连接没问题的话，就可以同步到GitHub啦！

用 Vercel 加速博客

由于GitHub访问速度过慢，我们需要使用Vercel加速。

1.登陆 Vercel

直接使用 GitHub 登陆即可。

2.导入 GitHub 仓库

打开 https://vercel.com/import/git 导入 GitHub 仓库。

3.绑定域名

仓库导入后，便可在项目设置中绑定域名或者使用Vercel提供的二级域名。

结束

按照上面做完以后，不出意外的话，你就能得到一个地址为xxx.vercel.app的博客啦，并且可以通过https访问，国内速度还超快哦。
